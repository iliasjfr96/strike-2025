# STRIKE 2025 — application

Code du jeu (client Vite/React/Three.js + serveur Node WebSocket). Voir le [README principal](../README.md) pour la présentation, le lancement et le déploiement.

- `src/` — client (rendu, prédiction, UI, éditeur de map)
- `server/` — serveur autoritaire (simulation, salons, API, admin)
- `src/shared/` — code partagé client/serveur (physique, map, armes, protocole)
- `scripts/dev.mjs` — lancement développement (serveur + Vite)
